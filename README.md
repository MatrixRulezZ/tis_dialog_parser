Проект: Бот для уведомлений через Telegram

Этот проект представляет собой Docker-контейнеризированное приложение для отправки уведомлений через Telegram с использованием данных из Личного кабинета "Диалог".
Начало работы
Предварительные требования

    Установленный Docker

    Установленный Docker Compose

    Аккаунт в Telegram

    Данные для доступа к Личному кабинету "Диалог"

Установка и запуск

    Клонируйте репозиторий:

bash

git clone <ваш-репозиторий>
cd <папка-проекта>

    Создайте файл .env в корневой директории проекта со следующим содержимым:

env

BOT_TOKEN="ТОКЕН_БОТА_БОТФАЗЕР"
CHAT_ID="ВАШ_ID_ТЕЛЕГРАМ"
TIS_LOGIN="ЛОГИН_ОТ_ЛК_ДИАЛОГ"
TIS_PASSWORD="ПАРОЛЬ_ОТ_ЛК_ДИАЛОГ"

    Запустите контейнер:

bash

docker-compose up -d --build

Остановка приложения

Для остановки и удаления контейнера выполните:
bash

docker-compose down

Конфигурация
Получение учетных данных

    BOT_TOKEN: Получите у @BotFather в Telegram

    CHAT_ID: Используйте @userinfobot для получения своего ID

    TIS_LOGIN/TIS_PASSWORD: Ваши учетные данные от Личного кабинета "Диалог"


Важные заметки

    Никогда не commitьте файл .env в Git!

    Убедитесь, что все переменные окружения заполнены корректными значениями

    Приложение запускается в фоновом режиме (демон) благодаря флагу -d

Troubleshooting

Если возникли проблемы с запуском:

    Проверьте правильность всех переменных в .env

    Убедитесь, что Docker запущен и работает корректно

    Проверьте логи контейнера: docker-compose logs
