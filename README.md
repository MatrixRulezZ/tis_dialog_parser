# Telegram TIS Bot

Бот для работы с личным кабинетом TIS и уведомлений в Telegram.  
Проект использует Docker для быстрого развертывания и изоляции окружения.

---

## Требования

- Docker  
- Docker Compose  
- Учетная запись Telegram  
- Учетная запись личного кабинета TIS

---

## Установка и настройка

1. Клонируйте репозиторий:

```bash
git clone <URL_репозитория>
cd <Папка_проекта>
````

2. Создайте файл `.env` в корне проекта со следующими данными:

```env
BOT_TOKEN="ТОКЕН_БОТА_БотФазер"
CHAT_ID="ВАШ_ID_ТЕЛЕГРАМ"
TIS_LOGIN="ЛОГИН_ОТ_ЛК_ДИАЛОГ"
TIS_PASSWORD="ПАРОЛЬ_ОТ_ЛК_ДИАЛОГ"
```

> ⚠️ **Важно:** Никогда не публикуйте файл `.env` с настоящими данными в публичных репозиториях.

---

## Запуск проекта

Соберите и запустите контейнер в фоновом режиме:

```bash
docker-compose up -d --build
```

Чтобы остановить и удалить контейнер:

```bash
docker-compose down
```
